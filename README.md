# sgfmodelis
Dépôt de recette et suivi des tests pour le projet SIG - Rapports de bugs, validations et demandes d'amélioration

## Structure du dépôt

- **`.github/ISSUE_TEMPLATE/`** : Templates pour créer des issues GitHub
  - `bug-anomalie.md` : Template pour les rapports de bugs
  - `validation-fonctionnalite.md` : Template pour les validations
  - `demande-amelioration.md` : Template pour les demandes d'amélioration
  - `autre-retour.md` : Template pour les autres retours

- **`issues/`** : Documentation des bugs et problèmes identifiés
  - Contient les rapports détaillés des anomalies détectées
